# Deterministic Reflection Agent

This project is a structured reflection tool that helps users analyze their day using a fixed decision tree.

## What it does

* Asks structured questions
* Uses fixed options (no typing)
* Moves step-by-step based on answers
* Gives a reflection at the end

## Files

* `tree/reflection-tree.json` → main logic
* `tree/tree-diagram.png` → flow diagram

## Key Idea

This is NOT a chatbot.

It is a deterministic system:

* Same answers → same output
* No AI used at runtime

## Why

To ensure:

* consistency
* reliability
* no hallucination
